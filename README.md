## Thesis Topic
Comparative Study of Unreal Engine and Unity:
Capabilities, Game Development Implementation, and Guidance for New Developers.

## Description
This thesis presents a comparative analysis of Unreal Engine and Unity,
focusing on their capabilities, development workflows, performance,
and suitability for new developers.
Two game prototypes were implemented in both engines to evaluate
practical differences.
